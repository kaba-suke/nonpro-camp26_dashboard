# nopro-camp26 プロジェクト (Nonpro Camp 2026 申し込みダッシュボード)

## このリポジトリの目的
このリポジトリは、ノンプロ研が主催するイベント「ノンプログラマーズ・テックキャンプ 2026」の申し込み状況をリアルタイムで可視化するためのダッシュボードアプリケーションを管理することを目的としています。

目標（最低:100名、目標:150名、最大:200名）に向けた進捗をグラフやアニメーションで分かりやすく見せることで、運営メンバー全員が「現在地」を把握し、集客やフォローアップへの士気向上（ワクワク感・期待感）を生み出すことを目指しています。

---

## このリポジトリの使い方

このダッシュボードは Python のウェブフレームワーク **Streamlit** を使用して構築されています。

### 起動手順

1. **ターミナル（またはコマンドプロンプト）を開き、このプロジェクトのディレクトリ（`nopro-camp26`）に移動する**

2. **仮想環境内の Python を使って Streamlit を起動する**

   ```powershell
   .\.venv\Scripts\python.exe -m streamlit run dashboard.py
   ```

   > ※ `streamlit run dashboard.py` だけでは動作しない場合があります。必ず `.venv` 内の Python を指定して実行してください。

3. **ブラウザでダッシュボードを確認する**

   実行が成功すると、ターミナルに以下のようなメッセージが表示されます。

   ```
   Local URL: http://localhost:8501
   ```

   自動的にブラウザが開かない場合は、上記の URL にアクセスしてください。

### データの更新について

#### ✅ 現在のデータ取り込み方式（Googleスプレッドシート直接読み込み）

> **2026年5月12日より、データソースを `sample_data.csv` からGoogleスプレッドシートへ切り替えました。**  
> これ以降、CSVファイルの手動更新は不要です。

現在のダッシュボードは、**CSVファイルのアップロード不要**です。起動するだけで最新データが自動的に取り込まれます。

```
Peatix (注文データ) → Googleスプレッドシート → dashboard.py が自動読み込み
```

- ダッシュボードを起動すると、Googleスプレッドシートから最新の申し込みデータを自動で取得します。
- 取得したデータは **10分間キャッシュ（一時保存）** されます。
- 手動で最新データに更新したい場合は、ブラウザ画面の右上メニュー（`⋮`）から **「Clear cache」** を選択し、ページをリロードしてください。

#### 📋 切り替え前後の比較

| | 切り替え前（〜2026/5/11） | 切り替え後（2026/5/12〜） |
|---|---|---|
| データソース | `sample_data.csv`（ローカルファイル） | Googleスプレッドシート（自動取得） |
| 更新方法 | CSVを手動エクスポートして上書き | ダッシュボードを起動するだけ（自動） |
| リアルタイム性 | 手動更新のタイミングのみ | 起動時に最新データを取得（10分キャッシュ） |

#### 📝 確認用CSVの出力（任意）

データクレンジング後の内容をファイルで目視確認したい場合は、専用のスクリプト `export_data.py` を実行します。

```powershell
.\.venv\Scripts\python.exe export_data.py
```

実行すると、プロジェクトフォルダ内に `csdata.csv` が生成されます。Streamlitを起動せずに、クレンジング結果だけを素早く確認したい場合に便利です。

## このリポジトリで行っている処理の概要

### 1. データの取得とクレンジング

- Googleスプレッドシートの公開URLからCSV形式でデータを自動ダウンロードして読み込みます。
- 以下の条件を **両方** 満たすレコードを除外します（重複注文の補助チケットを取り除くため）:
  - 「注文番号」が他の行と重複している
  - 「チケット名」に `イベント参加チケット【無料】` または `投げ銭で応援` が含まれる

### 2. 集計用データの加工

- 「属性」列を元に、参加者の所属を以下の3区分に分類した「コミュニティ」列を作成します。

  | 属性の値 | コミュニティ区分 |
  |---|---|
  | ノンプロ研のメンバー | ノンプロ研 |
  | OB・OG・元 が含まれる | 元ノンプロ研 |
  | それ以外 | その他 |

- 「枚数」列を数値データに変換し、総申込数・コミュニティ別人数を算出します。
- 「申込日」列を日付型に変換し、日別・累計の申し込み件数を算出します。

### 3. KPIの表示と演出

- 総申込数が目標値に到達した際には、以下のアニメーションとメッセージを表示します。
  - 100名達成 → ✅ 通知メッセージ
  - 150名達成 → 🔥 通知メッセージ
  - 200名達成 → 🎉 紙吹雪（バルーン）アニメーション

### 4. グラフによる可視化（Altair）

- 画面を開いた際に、グラフがアニメーションしながら表示されます。
  - **目標進捗ドーナツグラフ**: 現在の申込数と、100名・150名・200名の各目標に対する残り枠を可視化。
  - **参加者内訳ドーナツグラフ**: ノンプロ研・元ノンプロ研・その他の割合を可視化。
  - **申込者数推移グラフ**: 日別申込数（棒グラフ）と累計申込数（折れ線）の複合グラフ。目標ライン（100/150/200名）も表示。

## 各ファイル・フォルダの役割

| ファイル/フォルダ | 役割 |
|---|---|
| `dashboard.py` | ダッシュボードのメインスクリプト。データ取得・クレンジング・集計・グラフ描画のすべてのロジックが含まれる。 |
| `export_data.py` | クレンジング後のデータをCSV出力するための確認用スクリプト。 |
| `sample_data.csv` | **昨年（2025年）のキャンプ申し込みデータ**を参考として保存したファイル。2026年5月12日以降の本番運用では使用しない。過去データの傾向確認や開発・テスト時の参照用として残してある。 |
| `csdata.csv` | データクレンジング後の確認用CSV。 `export_data.py` を実行することで生成される。 |
| `steering_file.md` | ダッシュボード開発の目的・要件定義・実装機能などをまとめた仕様書。 |
| `.venv/` | Pythonの仮想環境フォルダ。Streamlit・Pandas・Altairなど必要なライブラリが格納されている。 |
